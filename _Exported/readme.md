## Contents

* **`SketchIO_Final_Submission_v1.1.2a.zip`:** The direct exported project in compressed ZIP format, generated from Android Studio.
* **`SketchIO_Release_APK_v1.1.2a.apk` (Optional):** The final signed Android Package Kit (APK) file for direct installation on a device.

## Purpose

This ZIP file includes all necessary source code, resource files, and configuration scripts (`build.gradle`, etc.) required to compile and run the final application successfully in a separate Android Studio environment. It fulfills all requirements, including the implemented **Fragment-based architecture**, **advanced saving/printing logic**, **custom drawing canvas**, and **full resource integration** (custom colors, strings, drawables).
