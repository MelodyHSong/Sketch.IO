# ☆ _Exported Directory ☆

This directory contains the final, complete project source code prepared for submission and archiving.

## Contents

* **`SketchIO_Final_Submission.zip` (or similar file name):** The direct exported project in compressed ZIP format, generated from Android Studio.

## Purpose

This ZIP file includes all necessary source code, resource files, and configuration scripts (`build.gradle`, etc.) required to compile and run the application successfully in a separate Android Studio environment, fulfilling the final assignment requirement.
